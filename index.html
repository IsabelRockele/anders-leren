Welkom !

Hier beheer je je klas, kies je per leerling de woorden en maak je werkbladen.

Hoe werkt het?

1

👥 Voeg een leerling toe

Maak een code aan in Mijn klas. Print de QR-code en geef ze mee naar huis. Zo logt het kind in zonder paswoord.

2

🏷️ Kies wat de leerling oefent

Klik bij een leerling op het 🏷️-knopje. Vink de thema's aan die het kind al kan, en binnen elk thema desgewenst per categorie of per woord.

💡 Tip: een nieuwe leerling start standaard met niets. Begin met enkel Klas (survival).

3

🏷️ Personaliseer woorden voor de hele klas

In Woorden kan je het basispakket aanpassen voor jouw klas: tekst wijzigen, eigen foto's gebruiken, of nieuwe woorden toevoegen. Geldt voor alle leerlingen.

4

📊 Volg de voortgang

Bij Wie kan wat? zie je in één oogopslag welke woorden elke leerling al kent (3 sterren) of nog moeilijk vindt.

5

📄 Maak werkbladen

Genereer PDF-werkbladen per thema, met 10 verschillende oefenvormen. Inclusief oplossingssleutel.

👁️

Wil je zien wat het kind ziet?

Rechtsboven staat een knop "Bekijk kind-app". Of klik op het 👁️-knopje naast een leerling om als die leerling in te loggen.

⚠️ Firebase nog niet ingesteld

Vul eerst je Firebase-config in firebase-config.js in. Daarna werkt dit paneel volledig.

ℹ️ Hoe werkt het?

  1. Maak een code per leerling in tabblad "Mijn klas". Schrijf op papier op welk kind welke code heeft (de cloud kent geen namen).
  2. Geef het kind de code om in te tikken op tablet of computer in de klas.
  3. Print de QR-code en geef die mee naar huis. Ouders scannen → kind logt automatisch in op zijn/haar voortgang.
  4. Bekijk in "Wie kan wat?" wie welke woorden al kent (3 sterren = "gekend").

🔒 Privacy & AVG

Alleen voortgangsdata (welk woord juist/fout) en de zelfgekozen code worden in de cloud opgeslagen. Namen blijven enkel op jouw paneel staan, niet bij de leerling-data. Geen e-mails, geen wachtwoorden, geen persoonlijke gegevens.

⭐ Sterrensysteem

  • 0 sterren: nog niet getest
  • 1 ster: 1 keer juist beantwoord
  • 2 sterren: 3 keer juist op rij
  • 3 sterren: 5 keer juist op rij = "gekend"
